# `src/scouts/` — Machine-Readable Blueprint (MRB)

## Role Definition
This directory houses the Extractor Scouts for Deep Pulse. Scouts perform target navigation, parsing of external configurations, and standardized intelligence distillation. 

**Local Sub-Agent Permission Level**: HIGH CONFIDENCE REQUIRED
You are permitted to write new OSINT capability modules here, but you MUST strictly inherit from `BaseScout` and utilize the defined "Stub" system.

## The "Never" List
1. **NEVER modify the core Curiousity Loop rules inside `base_scout.py`.** All modification overrides should be done on your specific child class instantiation.
2. **NEVER process Ed25519 cryptography here.** Cryptography must strictly hook remotely into `src/core/identity.py`. 
3. **NEVER bypass Tor configurations.** Future custom scouts must incorporate the `proxy_config` routing logic mapped in `web_scout.py`.
4. **NEVER write heavy library logic natively.** Utilize API endpoints over large dependency clones.
